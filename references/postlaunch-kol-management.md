# KOL 管理与发展指南

> 供 `drug-medical-postlaunch` Skill 在 L2(KOL管理与发展) 环节引用。
> 用途：统一 KOL 分层、画像、关系建设、洞察收集的方法论与模板。

## 1. KOL 分层框架

### 1.1 三层分级标准

| 层级 | 标准 | 互动策略 |
|---|---|---|
| **Tier 1** | 全国指南执笔人 / 主委级 / 头部学会任职 / 高影响力发文 | 深度合作：指南纳入 / 顾问委员会 / 学术发表 / 科学代言人 |
| **Tier 2** | 省级主委 / 区域学术带头人 / 区域学会任职 | 区域覆盖：区域学术活动 / 病例分享 / 科室影响力建设 |
| **Tier 3** | 活跃中青年骨干 / rising stars / 近年发文上升 | 培养路径：学术支持 / IIS 合作 / 会议演讲机会 / 长期关系培养 |

### 1.2 分层维度
- 学术影响力：发文(数量/期刊级别/被引) / 学会任职(级别/数量) / 指南参与(执笔/审稿/讨论)
- 临床影响力：科室地位 / 患者量 / 处方量 / 区域影响力
- 观念立场：支持 / 中立 / 质疑（须动态追踪，会变化）
- 竞品关系：是否为竞品 KOL / 与竞品的合作深度
- 学术方向：与产品适应症的匹配度

## 2. KOL 画像模板

### 2.1 画像字段
```
KOL 画像
├── 基本信息：姓名 / 机构 / 科室 / 职称
├── 学术档案
│   ├── 发文记录（代表性论文 / 近3年发文趋势 / 期刊级别）
│   ├── 学会任职（现任 / 历史）
│   ├── 指南参与（执笔 / 审稿 / 讨论）
│   └── 学术方向（主要研究方向 / 与产品匹配度）
├── 临床实践
│   ├── 科室地位（主任/副主任/带组）
│   ├── 患者量（年门诊量/住院量/手术量，估算标💭）
│   └── 处方习惯（治疗偏好/对新方案的接受度）
├── 观念立场
│   ├── 对本产品的态度：支持 / 中立 / 质疑
│   ├── 对竞品的态度
│   └── 观念变化追踪（上次互动→本次变化）
├── 合作历史
│   ├── 已合作项目（顾问会/IIS/演讲/发表）
│   └── 合作深度评估
├── 互动计划
│   ├── 下一阶段目标
│   └── 具体行动与时间
└── 动态更新（每次互动后更新画像）
```

### 2.2 画像维护要求
- 画像不是静态档案，是动态追踪工具——每次互动后更新观念立场和合作进展
- 观念变化是关键信号：从质疑→中立→支持是成功路径
- 发文和学会任职变化也要追踪（晋升/Tier 升级）

## 3. 关系建设路径

### 3.1 三阶段路径
```
阶段1：学术接触（冷启动）
  → 目标：建立学术信任，KOL 认可你的专业能力
  → 方式：学术拜访(科学数据交流) / 会议接触 / 病例讨论
  → 标志：KOL 愿意继续交流 / 主动提出问题

阶段2：科学合作（深化）
  → 目标：从"交流"到"合作"，建立共同产出
  → 方式：IIS 合作 / 顾问委员会 / 联合发表 / 会议演讲
  → 标志：KOL 主动参与项目 / 提出研究想法

阶段3：观念影响（成熟）
  → 目标：KOL 成为产品的科学代言人
  → 方式：指南纳入建议 / 学术会议发声 / 同行影响
  → 标志：KOL 在公开场合正面评价产品 / 主动向同行推荐
```

### 3.2 关系建设红线
- 不能用利益交换观念（合规底线）
- 学术合作须有真实科学价值（不是"请人站台"）
- IIS 须有科学价值而非变相推广
- 保持科学独立性：可以接受 KOL 对产品的质疑并反馈

## 4. 洞察收集结构化

### 4.1 洞察维度
| 维度 | 收集内容 | 策略价值 |
|---|---|---|
| 诊疗习惯 | 实际处方路径 / 药物选择逻辑 / 剂量调整习惯 | 理解真实临床决策 |
| 未满足需求 | 现有治疗的不满 / 期望的新方案特征 | 产品定位和拓展方向 |
| 竞品评价 | 对竞品的看法 / 使用体验 / 换药考虑 | 竞品策略调整 |
| 指南风向 | 即将更新的指南内容 / 专家对新数据的看法 | 提前布局指南纳入 |
| 准入反馈 | 医保/集采/药事会的实际影响 | 准入策略调整 |

### 4.2 洞察记录格式
```
洞察记录
├── 日期 / KOL / 场景(拜访/会议/电话)
├── 洞察内容（结构化记录，区分事实与KOL观点）
├── 信号强度（个案 / 多个KOL一致反馈 → 趋势）
├── 策略影响（是否影响医学策略/推广策略/证据生成）
└── 后续行动（是否需要进一步验证 / 反馈策略层）
```

### 4.3 洞察汇总机制
- 月度/季度汇总：多个 MSL 的洞察整合 → 识别趋势
- 趋势信号：单个 KOL 的观点可能是个案，3+ KOL 一致反馈是趋势
- 反馈路径：一线洞察 → 结构化汇总 → 医学策略层 → 影响 CDL/证据计划

## 5. Rising Stars 识别与培养

### 5.1 识别标准
- 年龄/职级：中青年（副教授/副主任医师级别，3-8 年内可能晋升）
- 学术上升势头：近 3 年发文量上升 / 开始参与学会 / 获得研究基金
- 临床影响力上升：科室内部话语权增加 / 患者量增长
- 与产品领域匹配：研究方向与产品适应症高度相关

### 5.2 培养策略
- 学术支持：提供最新研究数据 / 协助文献检索 / 学术讨论
- IIS 合作：支持其发起的研究（科学价值优先）
- 会议机会：推荐为学术会议演讲者 / 壁报展示
- 长期关系：3-5 年培养周期，在其晋升为 Tier 1 时关系已成熟

## 6. 常见陷阱
- KOL 画像做成静态档案不更新 → 失去追踪价值
- 关系建设跳过学术接触直接谈合作 → 信任基础不牢
- 洞察收集只记不汇总 → 一线信息到不了策略层
- 只维护 Tier 1 忽略 rising stars → 3-5 年后 KOL 梯队断档
- 用利益而非学术价值维系关系 → 合规风险 + 关系脆弱
- 把 KOL 当"推广工具" → 失去科学信任，KOL 疏远
