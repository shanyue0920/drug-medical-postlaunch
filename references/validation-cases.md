## 案例 1 · 环硅酸锆钠（利倍卓）高钾血症领域循证解读

### 1. 元数据头
- 分析对象：环硅酸锆钠散（Sodium zirconium cyclosilicate，商品名：利倍卓 / ZS-9）
- 适应症：成人高钾血症
- 分析视角：上市后医学 · 治疗领域洞察（L1）+ 循证解读（L2）/ 资深模式
- 数据截至：2026-07
- 证据等级总览：L1（NMPA 批准 + 全球试验）+ L2（2022 ACC/AHA/HFSA 指南）

### 2. 执行摘要
- 🔍 中国近 60 年**首个高钾血症创新药**，2020-01 获 NMPA 批准（2019 上半年入第二批《临床急需境外新药名单》，优先审评）。
- 🔍 起效快：服药后 1.0 小时血钾下降，恢复正常中位时间 2.2 小时，92% 患者 48h 内达正常（🔍 全球试验 + 中国药效学研究）。
- 🔍 长期控钾：HARMONIZE 显示用药 1 年 88% 患者血钾 ≤5.1 mmol/L，且可维持 RAASi 剂量（🔍 指南引用）。
- 💭 竞品帕替罗默（Patiromer）国内未上市，SPS（聚苯乙烯磺酸钠/钙）为旧方案、耐受性差。

### 3. 主体（L1 / L2）
- **L1 领域全景**：高钾常见于心衰/CKD 服用 RAASi 患者；未满足需求=快速降钾 + 长期维持 + 不中断 RAASi。缺陷：旧方案（SPS）起效慢、耐受差。
- **L2 循证解读**：
  - 指南 ≥5 篇可达（🔍 2022 ACC/AHA/HFSA 心衰指南：RAASi 后高钾用新型钾结合剂 IIb 类；中国心衰指南 2023 等）。
  - 关键研究：ZS-9 Ⅲ期（起效/正常化）、HARMONIZE（长期维持 RAASi）— 须 WebFetch 原文提取 ORR/血钾达标率/AE（💭 摘要级，建议原文核实）。
  - 推广数据点候选：①"1 小时起效"②"48h 92% 正常"③"可维持 RAASi"（✅ 合规）。
- **L4 优势**：起效速度、长期维持 RAASi 能力；弱点：长期用药成本/可及性（💭）。

### 4. 关键结论与行动建议
- 定位："快速降钾 + 长期维持 RAASi 的新型钾结合剂"。
- 建议：规划真实世界 RAASi 维持率研究；KOL 传递聚焦心衰/CKD 合并高钾场景。

### 5. 不确定性·待核实清单
- ⚠️ 待核实：ZS-9 Ⅲ期/HARMONIZE 具体 AE 频率与停药率（💭 指南仅给疗效数，建议查原文）。
- ⚠️ 待核实：中国获批精确日期（2020-01 多家报道，建议查 NMPA 批件号）。

### 6. 信息来源与证据等级
- 🔍 NMPA 批准（2020-01，阿斯利康公告 / 健康时报），L1。
- 🔍 2022 ACC/AHA/HFSA 心衰指南（ZS-9 推荐），L2。
- 💭 全球试验数据（行业报道），L3。

## 案例 2 · 司美格鲁肽（诺和泰）安全性信号医学应对

### 1. 元数据头
- 分析对象：司美格鲁肽（Semaglutide，诺和泰 / Wegovy / 诺和盈）
- 适应症：2 型糖尿病 / 肥胖 / 心血管代谢（多适应症）
- 分析视角：上市后医学 · 安全性信号应对（L2/L4/L11）/ 资深模式
- 数据截至：2026-07
- 证据等级总览：L1（FDA label 黑框）+ L2（白皮书中枢流行病学）

### 2. 执行摘要
- 🔍 美国标签含**甲状腺 C 细胞肿瘤（MTC）黑框警告**：有 MTC 个人或家族史 / MEN2 者禁用（🔍 诺和诺德 2026 新闻 + 清华药事说明）。
- 🔍 临床前（小鼠/大鼠终身暴露）显示 C 细胞肿瘤剂量/时间依赖性升高，但**人类相关性未确立**（🔍 说明书原文）。
- 💭 近期白皮书中枢流行病学（BMJ 北欧队列、Thyroid 多库分析）未发现 GLP-1 RA 显著增加常见甲状腺癌风险，提示存在检测偏倚；但结论限于 PTC/滤licular，不直接外推至 MTC。
- 🔍 常见 AE：恶心/腹泻/呕吐等胃肠道反应（🔍 说明书）。

### 3. 主体（L2 / L4 / L11）
- **L2 安全性全量**：黑框（MTC）为"需主动说明的安全信息"；胃肠道 AE 为常见可控；⚠️ 间接对比局限：啮齿类 vs 人类 C 细胞 GLP-1R 表达差异大。
- **L4 优劣势**：优势=强减重/心肾获益；弱点=甲状腺黑框顾虑 + 胃肠道耐受。
- **L11 合规审查**：
  - 核心信息"显著减重/心肾获益"✅ 可讲（有 RCT 支撑）。
  - "无甲状腺风险"❌ 不能讲（黑框仍在，人类数据未推翻）。
  - "黑框可能过度"⚠️ 慎讲（仅限医学科学交流，非推广话术）。
- **不利数据必须呈现**：黑框与 MTC 禁忌须如实传达，不得淡化。

### 4. 关键结论与行动建议
- 医学策略：以心肾/代谢获益为主轴，黑框作标准安全提示。
- 建议：MSL 拜访准备 MTC 黑框 FAQ；不将"白皮书中枢安全性"用于推广削弱黑框。

### 5. 不确定性·待核实清单
- ⚠️ 待核实：中国说明书是否含相同黑框及具体表述（💭 推断同 FDA，建议查 NMPA 说明书）。
- ⚠️ 待核实：白皮书中枢研究是否经同行评审发表及期刊名（💭 媒体报道，建议溯源）。

### 6. 信息来源与证据等级
- 🔍 诺和诺德中国新闻（2026-06-08，黑框声明），L1。
- 🔍 清华药事司美格鲁肽说明书（禁忌/MTC 警告），L1。
- 💭 Endocrine News / CancerNetwork 白皮书中枢分析，L3/L2（待同行评审确认）。

## 案例 3 · 达格列净（DAPA-HF）心衰领域循证解读与证据生成

### 1. 元数据头
- 分析对象：达格列净（Dapagliflozin）
- 适应症：射血分数降低心衰（HFrEF），无论是否合并糖尿病
- 分析视角：上市后医学 · 循证解读（L2）+ 证据生成（L6）/ 资深模式
- 数据截至：2026-07
- 证据等级总览：L1（DAPA-HF 顶刊 RCT）+ L2（中国心衰指南 2023）

### 2. 执行摘要
- 🔍 DAPA-HF（2019）：达格列净 vs 安慰剂，HFrEF 心血管死亡/心衰住院主要终点风险降 26%（🔍 心脏病学实践 2022 引用）；伴/不伴糖尿病获益相当（25%/27%）。
- 🔍 中国心衰指南 2023：有症状 HFrEF 无论糖尿病均推荐 SGLT2i（达格列净或恩格列净），Ⅰ类 A 级证据（🔍 指南精简版）。
- 💭 达格列净已拓展 HFpEF（DELIVER）与 CKD（DAPA-CKD），跨心肾获益。
- 🔍 SGLT2i 列入心衰"新四联"药物（💭 专家共识）。

### 3. 主体（L2 / L6）
- **L2 循证解读**：指南 ≥5 篇易满足（ACC/AHA/HFSA 2022、中国心衰指南 2023、ESC 等）；关键研究 DAPA-HF（RCT，L1）+ DELIVER（HFpEF）+ DAPA-CKD。推广数据点：①"主要终点降 26%"②"无论糖尿病均获益"（✅ 合规）。
- **L4 优势**：硬终点获益、跨糖尿病人群、肾心双护；弱点：eGFR<20 禁忌、症状性低血压。
- **L6 证据生成缺口**：真实世界 SGLT2i 在基线低血压/老年/联合"新四联"启动顺序的证据；基层可及性 RWE。
- **L11 合规**：数据点须引 DAPA-HF 具体 HR/CI，不得夸大为"治愈"。

### 4. 关键结论与行动建议
- 定位："心衰基石药物，心肾双护，跨糖尿病人群"。
- 建议：规划真实世界"新四联"序贯启动 RWE；准入价值证据包突出 26% 硬终点降幅。

### 5. 不确定性·待核实清单
- ⚠️ 待核实：DAPA-HF 具体 HR 值与 95%CI（💭 仅知降 26%，建议查 NEJM 原文）。
- ⚠️ 待核实：中国获批 HFrEF 适应症精确日期（💭 推断随指南，建议查 NMPA）。

### 6. 信息来源与证据等级
- 🔍 DAPA-HF（2019，L1 顶刊 RCT）。
- 🔍 国家心力衰竭指南 2023（精简版），L2。
- 💭 心脏病学实践 2022（DAPA-HF 解读），L2。
